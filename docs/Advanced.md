# Advanced Blocks

Advanced blocks give low-level control. Use with care.

## force cart velocity
Overrides easing and directly sets velocity.

## restore normal cart movement
Returns control back to easing-based movement.

## disable easing temporarily
Turns off easing for sharp movement.

## enable easing
Re-enables easing.

## is cart easing enabled
Returns true if easing is enabled.

## force finish ride
Immediately ends the ride.

## teleport cart
Moves the cart instantly to a position.

## raw velocity override active
Returns true if raw velocity is enabled.

## active cart sprite
Returns the cart sprite being controlled.

## cancel progress event
Removes a progress event at a specific percent.
