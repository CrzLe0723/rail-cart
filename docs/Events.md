# Setup Blocks

Setup blocks configure how the cart behaves before or during a ride.

## set cart base speed boost
Sets the normal movement speed and the boosted speed used with easing.

## set easing type
Chooses how the cart accelerates and decelerates.
Options include Sine, Linear, Cubic, Ease In, Ease Out, and more.

## is cart currently moving
Returns true if the cart is currently active.

## set cart sound
Plays a looping or background sound while the cart is active.

## follow cart with camera
Makes the camera follow the cart sprite.
