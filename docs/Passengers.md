# Passenger Blocks

Passenger blocks let multiple sprites ride the cart.

## add passenger
Adds a sprite to the cart.

## detach passenger
Removes a specific passenger from the cart.

## remove all passengers
Clears all passengers.

## cart has passengers
Returns true if at least one passenger is present.

## number of passengers on cart
Returns how many passengers are riding.

## teleport passenger
Moves a passenger independently.

## is sprite on the cart
Returns true if the sprite is riding the cart.
